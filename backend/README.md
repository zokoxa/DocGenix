# DocGenix

**AI-powered software project documentation generator.**

DocGenix interviews you about your project idea and automatically generates a complete documentation suite using 8 specialized AI agents, reviewed by a Critic agent.

## What it does

1. **Interviews you** — asks focused questions to gather your project name, platform, features, tech stack, and target audience
2. **Generates 8 documents** in parallel:
   - Project Overview
   - Requirements
   - User Stories
   - System Architecture
   - API Spec
   - Data Model
   - DevOps & Deployment
   - Testing Strategy
3. **Critic review** — automatically reviews and revises documents for consistency, completeness, and accuracy

## How to use

Just describe your project idea. DocGenix will guide you through a short interview, then generate your full documentation suite.

**Example prompts:**
- "I want to build a mobile app for food delivery"
- "Help me document my SaaS analytics platform"
- "Generate docs for a real-time chat application using React and Node.js"
- "Create documentation for my e-commerce website"
